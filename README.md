# Dashboard Atuarial - ANS

Projeto desenvolvido com base nos dados da Agência Nacional de Saúde Suplementar (ANS).

## Objetivo
Realizar análise atuarial do setor de saúde suplementar, com foco em sinistralidade,
custos e sustentabilidade financeira.

## Base de Dados
Dados públicos da ANS contendo:
- Operadoras
- Beneficiários
- Receita
- Despesa assistencial

## Métricas
- Sinistralidade
- Margem operacional
- Custo médio por beneficiário

## Visualizações
- Receita vs Despesa
- Margem operacional
- Beneficiários vs Receita
- Custo médio
- Sinistralidade
